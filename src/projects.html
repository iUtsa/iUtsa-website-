Edith GPT

Unlike traditional calendars, Edith not only schedules tasks but also takes into account your health, stress levels, and personal preferences to offer a more balanced and personalized experience. By integrating with multiple APIs, it can suggest calming Spotify playlists based on your stress levels, plan academic tasks, and even offer horoscope-based guidance. Edith also provides conversational planning options and suggests activities based on real-time weather updates, making it more than just a calendar—it's your intelligent personal assistant for better time management and overall well-being.

Technologies : HTML5,CSS3,JavaScript,Firebase,ExpressJS,NodeJS,React,Typescript,Git

Bank Portfolio Manager

The Bank Portfolio Manager is a Java-based desktop application built using Java Swing and Swing trees to provide an intuitive interface for managing bank accounts, transactions, and loans. It allows users to easily create and manage multiple accounts, track deposits and withdrawals, and monitor loan applications and repayments, all in real time. With a user-friendly design and secure data handling, this application simplifies financial management by offering a comprehensive overview of your entire banking portfolio in one place.

Technologies : Java, Java Swing, Eclipes, Git, GitHub

Fifa Player Stats App

FIFA Player Stats Viewer, a desktop application designed for football fans to easily access detailed statistics and information about FIFA players. With an interactive gallery of player images, users can simply click on a player's picture to view comprehensive stats, recent performances, and biographical details. Built with Java Swing, the app offers a smooth and user-friendly interface, ensuring an engaging experience. All player information is consistently updated to provide accurate and current data.

Ideal for students, educators, and anyone interested in visualizing mathematical data.
